One batch script touches several files at once — same timestamp on all of them. That is normal.

=== Default workflow (same idea as Combined: Mapping, no folder paths in the CSV) ===

  rest_of_repo_renamer_mapping.csv
    -> ReNamer rule: Mapping. Match = file basename. "Skip extension" OFF.

  rest_of_repo_renamer_mapping_skip_ext.csv
    -> Same, but Mapping "Skip extension" ON (Match/New Name without .ext).

  If a basename would map to more than one target, it is omitted from these files;
  see rest_of_repo_basename_mapping_skipped.txt.

=== Optional: Import menu (different from Mapping) ===

  Files toolbar -> Export -> "Import file paths and new names"

  renamer_no_combined_all_videos_import.csv
    -> Full tree; column 1 = path (absolute when generated by default).

  renamer_no_combined_import_paths_newnames.csv
    -> Matched-only, same Import format.

Use this path if Mapping is not enough (e.g. basename clashes). If Import matches nothing,
see RENAMER_ZERO_MATCHES.txt (paths / drive letter vs UNC).

=== Audit (not for Mapping or Import) ===

  rest_of_repo_rename_matched.csv
    -> OldRel, NewRel, Source — for spreadsheets / checking the map.
